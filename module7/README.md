## ğŸ“„ Files

| Exercise | Folder | File                     | Description                                                                                     |
| -------- | ------ | ------------------------ | ----------------------------------------------------------------------------------------------- |
| 00       | ex00/  | `parameter_matching.py`  | Compares user input with a single parameter; shows "Good job!" or "Nope, sorry..." or "none".  |
| 01       | ex01/  | `count_it.py`            | Shows the total number of parameters and each parameter with its length; shows "none" if empty.|
